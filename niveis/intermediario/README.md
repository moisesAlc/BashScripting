## Ordem sugerida (quando tudo estiver nesta pasta)

1. Condicional → Cálculos → Case → Loop (`for`, `while`)
2. Prática: [calculadora](../../atividades/01.calculadora/calculadora.sh) *(será movida para `pratica/calculadora/`)*
