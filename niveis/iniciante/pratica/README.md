# 🛠️ Prática — iniciante

Exercícios leves após `shell`, `entrada_usuario` e `pipe`. Ainda sem `if` complexo ou loops (isso é no intermediário).

## 📌 Sugestões

1. ⌨️ **Saudação** — script que usa `read -p` e imprime uma mensagem com o nome digitado.
2. 🔗 **Pipeline simples** — na pasta `pipe/`, rode `bash script.sh` e altere o filtro em `log.txt` (por exemplo trocar `ERRO` por outra palavra).
3. 🔗 **Contagem** — `cat log.txt | wc -l` e interprete a saída (reforço de pipe sem script novo).

## 📌 Referência

Os scripts em `../pipe/` já demonstram pipeline com `grep`, `sort` e `uniq`.
